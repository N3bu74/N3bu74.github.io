恺撒密码#

​ 在密码学中，恺撒密码（英语：Caesar cipher），或称恺撒加密、恺撒变换、变换加密，是一种最简单且最广为人知的加密技术。它是一种替换加密的技术，明文中的所有字母都在字母表上向后（或向前）按照一个固定数目进行偏移后被替换成密文。例如，当偏移量是3的时候，所有的字母A将被替换成D，B变成E，以此类推。

仿射密码#

仿射加密要求gcd(a,26)=0,否则就褪化为 y=f(k)=b(mod 26)。

加密函数： $$ m = D(x) = (ax+b) (mod\ n) $$ 解密函数： $$ c = E(y) = a^(y-b) (mod\ n) $$

这里a*是a关于n的乘法逆元。

欧几里得算法

​ 欧几里得算法（Euclid’s Algorithm）又称辗转相除法。古希腊数学家欧几里得在其著作 The Elements 中最早描述了这种算法，所以该算法被命名为欧几里得算法。算法利用公式 gcd(a,b) = gcd(b, a mod b)，直到a mod b为0时。a 就是原来两个非负整数 a 和 b 的最大公约数。

#递归
def gcd(a, b):
    if b == 0:
        return a
    else:
        return gcd(b, a % b)
#非递归
def gcd(a,b):
    while ( b != 0):
        a, b = b, a % b
    return a

扩展欧几里得算法

​ 扩展欧几里得算法（Extend Euclid’s Algorithm）是欧几里得算法的扩展，基于以下定理：对于任意两个整数 a, b，必存在整数 x, y，使得 ax + by = gcd(a,b) 成立，求出整数解 x, y，可以得到 gcd(a,b)。

​ 在欧几里得算法中，终止状态是b == 0时，这时候其实就是gcd(a,0)；我们想从这个最终状态反推出刚开始的状态。由欧几里得算法可知。gcd(a,b) = gcd(b,a mod b);那么有如下表达式：

gcd(a,b) = a*x1+b*y1;

gcd(b,a mod b) = b*x2+(a - a//b*b)*y2；（此处的a//b表示整除，例如：6//4 = 1；所以：a mod b = a % b = a - a//b*b）

化简上式有：b*x2 + a * y2 - a//b*b*y2

		= a*y2 + b*(x2 - a//b*y2)

与原式 gcd(a,b) = a*x1+b*y1; 对比，容易得出：

x1 = y2;
y1 = x2 - a//b*y2;

根据上面的递归式和欧几里得算法的终止条件b == 0,我们可以很容易知道最终状态是a * x1 + 0 * y1 = a;
即递归基 x =1 ,y =0

乘法逆元

​ 对于ax mod p = 1,这时，x就是a 在mod p乘法群的乘法逆元, 通过拓展欧几里得算法，得到ax + by = gcd（a，b）,因为a属于模b乘法群，所以a<b，所以a与b互素，则有gcd(a,b)=1,即 ax + by = 1,有了这个，我们就可以两边求mod b,即有 ax = 1（mod b），即此时的x就是乘法逆元。

​ 根据乘法逆元的性质，只有当a与b互素，a关于模b的乘法逆元有解。如果时不互素，则无解。那么这个方程就是a,b互质的充要条件是方程ax+by = 1必有整数解

# 扩展欧几里得算法
def ext_gcd(a, b):
    if b == 0:         
        return 1, 0, a     
    else:         
        x, y, q = ext_gcd(b, a % b)   # 递归直至余数等于0(需多递归一层用来判断)
        # q = gcd(a, b) = gcd(b, a%b)         
        x, y = y, (x - (a // b) * y)    # 辗转相除法反向推导每层a、b的因子使得gcd(a,b)=ax+by成立    
        return x, y, q
# 求乘法逆元
def inverse(a: int, n: int):
    '''
    return: a 在 mod n 的逆元
    '''
    x, _ , gcd= ext_gcd(a,n)
    if (gcd != 1):
        raise ValueError(str(a) + " has no inverse in mod " + str(n))
    return (x+n)%n       #为了使余数一定为正数

class Affine():
    def __init__(self,Z):
        self.Z = Z

    # 扩展欧几里得算法
    def ext_gcd(self,a, b):
        if b == 0:
            return 1, 0, a
        else:
            x, y, q = self.ext_gcd(b, a % b)  # 递归直至余数等于0(需多递归一层用来判断)
            # q = gcd(a, b) = gcd(b, a%b)
            x, y = y, (x - (a // b) * y)  # 辗转相除法反向推导每层a、b的因子使得gcd(a,b)=ax+by成立
            return x, y, q

    # 求逆元
    def inverse(self,a: int, n: int):
        '''
        return: a 在 mod n 的逆元
        '''
        x, _, gcd = self.ext_gcd(a, n)
        if (gcd != 1):
            raise ValueError(str(a) + " has no inverse in mod " + str(n))
        return (x + n) % n  # 为了使余数一定为正数

    #D(x) = (a*x+b) (mod 26)

def encode():
    global a,b,CryptoText,txt
    if math.gcd(26,a) == 1:
        for i in txt:
            tmp = a * (ord(i)-97) + b
            CryptoText += chr((tmp % m) + 97)
        print("[+]密文为----->",CryptoText)
        return 0
    else:
        print("[-]加密失败，密钥 a 与 26 不互质,请重新输入a")

# 求乘法逆元
def extendEdeuclid():
    global a,m
    x1,x2,x3 = 1,0,m
    y1,y2,y3 = 0,1,a
    while True:
        if y3 == 0 :
            return 0
        if y3 == 1 :
            if y2<0:
                y2 = m+y2
            return y2
        q = int(x3/y3)
        t1,t2,t3 = x1-q*y1, x2-q*y2, x3-q*y3
        x1,x2,x3 = y1,y2,y3
        y1,y2,y3 = t1,t2,t3

# 解密
def decode():
    global PlainText,CryptoText,a,b,m
    tmp = extendEdeuclid()
    if tmp != 0:
        for i in CryptoText:
            PlainText += chr(tmp * (ord(i)-97-b) % m + 97)

        return 0
    else:
        print("[-]解密失败，密钥",a,"与",m,"没有逆元，请重新输入密钥a")
        return 1
